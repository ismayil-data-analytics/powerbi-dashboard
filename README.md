# Power BI Sales Dashboard Project

## Overview
This project analyzes business data to provide insights into cost, income, revenue, and sales trends over time.

## Key Metrics
- Total Cost by Category  
- Income Analysis  
- Monthly Trend Analysis  
- Class vs Revenue Analysis  

## Tools Used
- Power BI  
- DAX  
- Excel  

## Goal
To turn raw data into meaningful business insights.
